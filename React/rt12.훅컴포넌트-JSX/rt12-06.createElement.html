React.createElement() 를 사용하여 html 태그 생성 방법을 알아본다. JSX가 어떻게 JavaScript로 변환되는지 시험해보고 싶다면 온라인 babel 컴파일러를 사용하여 본다. <a href="https://babeljs.io/repl/#?presets=react" target="_blank" rel="noreferrer">온라인 babel 컴파일러</a> #wrap1 에 JSX 를 사용하여 화면을 만드시오 <h1 id="greeting"> <p> Test!!! wrap1 </p> </h1> #wrap2 에 React.createElement() 를 사용하여 화면을 만드시오. React.createElement("h1", { id: "greeting" }, React.createElement("p", null, "Test!!! wrap2" ) ) #wrap1 이나 #wrap2 에 출력되는 화면은 동일하다.
